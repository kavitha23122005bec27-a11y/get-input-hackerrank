# Simple Sum Calculator (C++)

This is a basic C++ program that reads three integers from the user and outputs their sum. It serves as a foundational example of using standard input/output streams and basic arithmetic in C++.

## Description

The program demonstrates the core mechanics of the C++ `iostream` library. It initializes three integer variables, captures user input via `cin`, and performs an addition operation. The result is then printed to the console using `cout`.

### Key Features
* **Standard I/O:** Uses `std::cin` and `std::std::cout` for data handling.
* **Efficient Logic:** Performs a single-pass calculation with $O(1)$ space complexity.
* **Clean Code:** Follows standard competitive programming boilerplates.
